# NOVA RIFT — Frontend (React Native)

Aplicación móvil de NOVA RIFT desarrollada en **React Native** con **Expo**.

## Requisitos previos

- [Node.js](https://nodejs.org/) (LTS) instalado y disponible en el PATH.

## Instalación y ejecución

1. Abrir una terminal en esta carpeta (`02_FRONTEND_MOBILE`).
2. Instalar dependencias: `npm install`
3. Arrancar la app: `npm start` o `npx expo start`

## Configuración y tema

La app usa un layout estilo eSports: fondo oscuro con acentos de neón (`#00e5ff` cyan). La comunicación con el backend se realiza con Axios contra el puerto 8080. Desde el emulador de Android se usa la dirección `10.0.2.2`, alias de `localhost` del equipo anfitrión.
